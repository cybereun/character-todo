# 캐릭터 할일 프로그램 제작 플랜

## 목표
- Windows 11 바탕화면 오른쪽 아래에 떠 있는 투명 배경 캐릭터 할일 앱을 만든다.
- 첨부 캐릭터 이미지를 기본 상태와 배가 부른 상태로 사용한다.

## 기술 선택
- Electron 데스크톱 앱으로 구현한다.
- 프레임 없는 투명 창을 사용해 캐릭터 위젯처럼 보이게 한다.
- 할일 데이터는 앱 내부 `localStorage`에 저장한다.

## 기능 계획
1. 앱 시작 시 작업 표시줄을 제외한 화면 오른쪽 아래에 캐릭터 창을 배치한다.
2. 캐릭터 클릭 시 창 크기를 위쪽으로 확장하고 할일창을 슬라이드/페이드 애니메이션으로 표시한다.
3. 할일 입력 후 Enter 또는 추가 버튼으로 즉시 목록에 추가한다.
4. 각 할일은 체크 완료, 수정, 삭제를 지원한다.
5. 체크 완료 시 효과음, 파티클, 캐릭터 바운스 효과를 실행한 뒤 항목을 제거한다.
6. 캐릭터 클릭 시 할일창을 다시 닫고 창 크기를 캐릭터 크기로 줄인다.
7. 캐릭터는 계속 둥실둥실 움직이고 주기적으로 눈 깜빡임 효과를 낸다.
8. 마우스 드래그로 캐릭터 창을 화면 어디든 이동할 수 있게 한다.
9. 할일이 하나라도 있으면 배부른 이미지로 전환하고, 할일이 0개면 날씬한 이미지로 되돌린다.

## 파일 구성
- `package.json`: Electron 실행 스크립트와 앱 메타데이터
- `src/main.js`: 창 생성, 위치 계산, 드래그 이동, 확장/축소 처리
- `src/preload.js`: 렌더러와 메인 프로세스 간 안전한 API
- `src/index.html`: 앱 UI
- `src/styles.css`: 투명 위젯, 캐릭터 애니메이션, 할일창, 효과 스타일
- `src/renderer.js`: 할일 로직, 저장, 수정/삭제/완료 효과, 입력 처리
- `assets/character-slim.png`: 기본 캐릭터 이미지
- `assets/character-full.png`: 할일이 있을 때의 캐릭터 이미지

## 검증 계획
- `npm install` 후 `npm run start`로 앱 실행 가능 여부를 확인한다.
- `npm run lint:js` 또는 Node 문법 검사로 주요 JS 파일을 확인한다.
